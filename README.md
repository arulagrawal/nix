## Nix configuration

My nix configuration for my macbook.

Using [mk-darwin-system](https://github.com/vic/mk-darwin-system/tree/main) using [my fork](https://github.com/arulagrawal/mk-darwin-system/).

### Usage:
 - Install on a new system with `make bootstrap`
 - Check if the config is good with `make test`
 - Format with `make format`
 - Install with `make install`
 - Update sources with `make update`
